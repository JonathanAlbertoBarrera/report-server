# Report Server

## Levantar el proyecto

**Backend**: Entra a la carpeta report-server, ejecuta npm install y luego npm run start:dev

**Base de datos**: Ejecuta docker-compose up -d

**Acceder a pgAdmin**: Ve a http://localhost:8080 y usa email: superman@google.com, password: 123456

**Configurar servidor en pgAdmin**: Host: postgres_database, Port: 5432, Username: postgres, Password: 123456
